torch-neuronx Documentation
===========================

torch-neuronx is a PyTorch backend for AWS Neuron devices. It uses PyTorch's `PrivateUse1 <https://docs.pytorch.org/tutorials/advanced/privateuseone.html>`_ mechanism to add 'neuron' device to PyTorch.
